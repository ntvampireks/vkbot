# Руководство по подключению бота к сообществу VK

Это пошаговое руководство поможет подключить бота к вашему сообществу VK.

## Предварительные требования

- Аккаунт VK с правами администратора сообщества
- Сообщество (группа или публичная страница)
- Python 3.11+

---

## Шаг 1: Создание приложения VK

1. Перейдите в [раздел разработчиков VK](https://dev.vk.com/)
2. Нажмите кнопку **«Создать приложение»**
3. Заполните форму:
   - **Название**: любое (например, «Бот сообщества»)
   - **Категория**: любая
   - **Платформы**: поставьте галочку **«Сообщества»**

4. Нажмите **«Сохранить»**

---

## Шаг 2: Настройка сообщества

### 2.1 Перейдите к управлению сообществом

1. Откройте ваше сообщество VK
2. Нажмите **«Управление»** (в меню справа или под аватаром)

### 2.2 Включите работу с API

1. В разделе **«Настройки»** найдите блок **«Работа с API»**
2. Нажмите **«Включить»** (если отключено)

---

## Шаг 3: Создание токена доступа

1. В разделе **«Работа с API»** нажмите **«Создать токен»**
2. Выберите права:
   - ✅ **messages** — доступ к сообщениям (обязательно)
   - ✅ **groups** — доступ к группам (если нужно)

3. Нажмите **«Создать»**
4. **Скопируйте токен** — он показывается только один раз!

> ⚠️ Сохраните токен в безопасное место. Не передавайте его и не коммитьте в git.

---

## Шаг 4: Подключение Long Poll

Long Poll позволяет получать новые сообщения в реальном времени.

1. В разделе **«Работа с API»** найдите блок **«Long Poll API»**
2. Нажмите **«Подключить»**
3. Система сгенерирует:
   - **server** — адрес сервера
   - **key** — ключ доступа
   - **v** — версия API

Эти данные используются для прямого опроса сервера VK.

---

## Шаг 5: Настройка бота

### 5.1 Скопируйте файл окружения

```bash
cp .env.example .env
```

### 5.2 Заполните переменные

Откройте `.env` и вставьте данные:

```env
# Токен из Шага 3
VK_GROUP_TOKEN=your_token_here

# ID сообщества (цифры, без минуса)
VK_GROUP_ID=12345678

# Опционально
BOT_HOST=localhost
BOT_PORT=8000
DIALOG_TIMEOUT_HOURS=24
```

### Как найти ID сообщества

**Способ 1:** В URL управления

```
https://vk.com/edit?act=manage&group=12345678
                               ^^^^^^^^^^ это ID
```

**Способ 2:** В настройках сообщества

1. Управление → Настройки
2. **Внутренний номер сообщества**: `12345678`

**Способ 3:** Через API

```bash
curl "https://api.vk.com/method/groups.getById?group_id=your_community_name&access_token=YOUR_TOKEN&v=5.131"
```

## Шаг 6: Установка и запуск

### 6.1 Установите зависимости

```bash
pip install -r requirements.txt
```

### 6.2 Запустите бота

```bash
python -m bot.main
```

Вы должны увидеть:

```
2026-05-11 15:30:00 - bot - INFO - Бот запущен...
```

---

## Шаг 7: Проверка работы

### 7.1 Настройте уведомления сообщества

1. Управление → Настройки
2. В разделе **«Управление доступом»** добавьте бота (если нужно)
3. В разделе **«Сообщения»** → включите **«Включено»**
4. Добавьте бота в **«Служба поддержки»** (если требуется)

### 7.2 Напишите тестовое сообщение

1. Откройте диалог с сообществом
2. Напишите: `привет`
3. Бот должен ответить: `Здравствуйте! Я бот сообщества VK. Чем могу помочь?`

## Возможные проблемы

### Ошибка: `Access token access was denied`

**Причина:** Токен не имеет нужных прав

**Решение:** Пересоздайте токен с правами `messages`

### Ошибка: `Invalid group_id`

**Причина:** Неправильный ID сообщества

**Решение:** Убедитесь, что используете числовой ID без минуса

### Сообщения не приходят

**Причина:** Long Poll не подключён или сообщество не разрешает боту читать сообщения

**Решение:**
1. Проверьте, что Long Poll подключён
2. В настройках сообщества включите **«ВКлючить сообщения»**
3. Убедитесь, что бот добавлен в **«Служба поддержки»**

## Тестирование через curl

Проверьте токен:

```bash
curl "https://api.vk.com/method/groups.getById?group_id=12345678&access_token=YOUR_TOKEN&v=5.131"
```

Отправьте сообщение вручную:

```bash
curl -X POST https://api.vk.com/method/messages.send \
  -d "peer_id=12345678" \
  -d "message=Привет!" \
  -d "random_id=1" \
  -d "access_token=YOUR_TOKEN" \
  -d "v=5.131"
```

## Структура .env

| Переменная | Описание | Пример |
|------------|----------|--------|
| `VK_GROUP_TOKEN` | Токен доступа сообщества | `vk123abc...` |
| `VK_GROUP_ID` | Числовой ID сообщества | `12345678` |
| `BOT_HOST` | Хост для запуска | `localhost` |
| `BOT_PORT` | Порт для запуска | `8000` |
| `DIALOG_TIMEOUT_HOURS` | Таймаут сессии (часы) | `24` |
